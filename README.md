## 🧠 Overview  

This project is not just a portfolio website — it is a **modern frontend system** built to present work in a structured, scalable, and visually clear way.

It focuses on:
- clear presentation of projects and their purpose  
- smooth and responsive user experience  
- structured navigation for fast evaluation  

The goal is to make it easy for anyone (especially recruiters) to quickly understand both **what I’ve built and how I think as a developer**.

---

## 🏗️ Architecture  

The application is designed with a modular frontend structure:

- **Component Layer** — reusable UI components  
- **Animation Layer** — GSAP-powered transitions and interactions  
- **State Management** — handled via React hooks  
- **Data Layer** — static, config-driven project rendering  

This structure ensures:
- scalability  
- maintainability  
- easy extension of features  

---

## ⚙️ Tech Stack  

- **Frontend:** React + TypeScript + Vite  
- **Styling:** CSS  
- **Animation:** GSAP (GreenSock Animation Platform)  
- **Deployment:** Vercel  

---
